# totte — わたしのWishList

欲しいものとタスクを、ブラウザの端末内保存だけで管理する個人用Webアプリです。ログイン、サーバー、インストールは不要です。

## いちばん簡単な起動方法

1. このフォルダを開きます。
2. `index.html` をダブルクリックします。
3. ブラウザで「totte」が開いたら、そのまま使えます。

入力内容は、開いたブラウザの中に自動で保存されます。同じ `index.html` を同じブラウザで開けば、前回の続きから使えます。

## 大切な注意

- ブラウザの閲覧データやサイトデータを削除すると、入力内容も消えることがあります。
- 別の端末や別のブラウザには自動で引き継がれません。
- 最初は使い方が分かるよう、サンプルの欲しいものとタスクが入っています。各カードの「…」メニューから削除・編集できます。

## 入っている機能

- 欲しいものの追加・編集・削除
- 商品画像の追加（大きな画像は端末内保存向けに自動で軽量化）
- カテゴリ、優先度、必要度、買う場所、メモ、商品リンクの記録
- カテゴリの追加・名前変更・削除
- 優先度ごとの合計金額
- 月ごとのコスメ・日用品予算
- 購入済み履歴と「欲しいものに戻す」操作
- 4つのタイミング別タスク、任意の締め切り日時、完了管理
- スマートフォン／パソコンの両方に合わせた表示

画像も他の入力内容と同じくブラウザ内に保存されます。画像をたくさん登録するとブラウザの保存容量に達することがあるため、必要な画像だけを登録するのがおすすめです。
